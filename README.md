# sage-mode 

This feature introduces a real-time **Sage Mode filter** inspired by Naruto Uzumaki from the Naruto anime series.

The filter activates automatically when the user closes their eyes for a short duration, recreating the iconic Sage Mode transformation.

---

## ✨ Features

### 👁️ Eye-Based Activation (EAR Method)
- Uses the **Eye Aspect Ratio (EAR)** technique to detect eye closure.
- When both eyes remain closed for **3 seconds**, Sage Mode is triggered.
- Helps prevent false activation from normal blinking.

---

### 🟡 Sage Mode Visual Effects
Once activated, the filter applies signature Sage Mode visuals:

- **Yellow Pupils**  
  Sharp, glowing yellow pupils inspired by Sage Mode.

- **Orange Eyelids / Eye Shadow**  
  Orange pigmentation around the eyelid and black eyelashed for an anime-accurate look.

- **Authentic Transformation Feel**  
  Designed to closely resemble Naruto’s Sage Mode appearance.


### 🔄 Auto Toggle System
- **Activate:** Close eyes for 3 seconds  
- **Deactivate:** Pressing the keywords S or s
